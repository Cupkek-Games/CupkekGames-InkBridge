# CupkekGames Ink

Ink narrative engine integration for Luna UI. Parses Ink stories, drives speech bubbles + choice popups via Luna components.

## What's inside

- **`CupkekGamesInk/`** (CupkekGames.Luna.Ink.asmdef) — Ink integration runtime: story controller base, narrative event bus, Luna UI binding helpers.

## External dependency

Requires the **ink-unity-integration** package from inkle (https://github.com/inkle/ink-unity-integration). Install separately.

## Dependencies

- `com.cupkekgames.keyvaluedatabase` (story character database)
- `com.cupkekgames.luna` (UI components for speech bubbles, choices)
- `com.inkle.ink-unity-integration` (third-party; install per inkle instructions)
